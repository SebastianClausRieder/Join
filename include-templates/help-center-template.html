What is Join?

Join is a visual system for managing work throughout a process.
Join visualizes both the process (the workflow) and the actual work that goes through that process.
The goal of Kanban is to identify and address potential bottlenecks in your process so work can proceed cost-effectively and at optimal speed or throughput.

Summary

Here you can find an overview about amount of tasks, how many tasks which status have and when is next urgent coming up task.

Add Task

Create new tasks with following parameters:

  • Title: give decisive title
  • Description: add description to explain details of task
  • Category: choose category of department or create new category
  • Assing to: choose members to assign the task to
  • Due date: pick latest date when task should be finished
  • Prioriy: Choose kind of priority of urgency
  • Add to Task: Determine in which task should be created, or create a completely new task.
  • Subtasks: Add new subtasks, so you can break the task down into multiple stages of completion

    • After creating task you can find it in the board.
